📚 如何取得ChatGPT API Key

🎥 教學影片

📖 步驟一：註冊OpenAI帳號

首先，您需要前往 OpenAI 官方網站註冊一個帳號。這是取得 API Key 的第一步驟。

重要提醒：註冊時請使用有效的電子郵件地址，因為需要進行電子郵件驗證。

註冊步驟：

1. 前往 https://platform.openai.com

2. 點擊「Sign up」建立新帳號

3. 輸入電子郵件地址和密碼

4. 驗證電子郵件

⚙️ 步驟二：設定付款方式

設定信用卡資訊：

要使用 OpenAI API，您需要設定付款方式。OpenAI 採用用量計費的方式，只有實際使用時才會收費。

付款設定步驟：

1. 登入 OpenAI 平台後，前往「Billing」頁面

2. 點擊「Add payment method」

3. 輸入信用卡資訊

4. 設定使用額度限制（建議先設定較低額度）

💡 步驟三：產生API Key

產生您的 API Key：

完成付款設定後，您就可以產生 API Key 了。這個 Key 就像是您應用程式的身分證，用來識別和授權 API 請求。

安全提醒：API Key 非常重要，請妥善保管，不要分享給他人！

產生 API Key 步驟：

1. 前往「API Keys」頁面

2. 點擊「Create new secret key」

3. 為 Key 命名（例如：我的應用程式）

4. 複製產生的 API Key 並安全保存

5. 在您的應用程式中使用這個 Key
