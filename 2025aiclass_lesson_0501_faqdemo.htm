3. 多元摘要

  • AI工具：ChatGPT、Google Gemini
  • 上傳：2023年永續報告書
  • 步驟1. 生成各種格式的摘要
prompt
你是一個專業文章編撰專家 
依我上傳的內容 
彙整成易懂、重點，並保留關鍵字及相關原文

一、首先深入閱讀並理解內容
二、請幫我整理各種重點摘要格式
(一)標題
(二)一句話摘要
(三)300字摘要：一段文字，說明重點內容
(四)重點條列摘要，至少有3層的內容
(五)請整理內容中的關鍵字(含原文)
  • 步驟2. 生成可分享的一頁式網頁(編寫或編碼)
prompt
直接將以上內容改為一頁式網頁Code(HTML + CSS + JS) 
記得不要偷懶，該生成的內容就該完整呈現
網頁可一直往下拉閱讀完內容、互動式卡片呈現易讀的環境讓使用者更易理解
包括以下幾個大項，以選單顯示：
- 重點摘要：含一句話摘要、300字摘要
- 條列重點：markdown code格式、有按鈕按下後可一鍵複製
- 心智圖重點：內容以Mermaid心智圖直接顯示在網頁中(引入 Mermaid 的 CDN以直接顯示，要確認是否能正常顯示)
- 關鍵字整理（含原文）
最上方固定位置標頭包括：標題、選單、可放大縮小文字的按鈕

4. 財報分析

  • AI工具：ChatGPT、Google Gemini
  • 上傳：Google 搜尋Nvidia財報
  • 步驟1. 生成財報重點摘要
prompt
你現在是一位頂級的、世界一流的金融分析師，專門為不懂金融的初學者解讀財報。  
請基於我上傳的這份 NVIDIA 的財報，用最通俗易懂的語言，為我提供一份摘要，**必須包含以下幾個部分**：

1. **總體評價**：用一句話總結這家公司本季的整體表現（優秀、良好、一般、差）。
    
2. **損益表分析**：關鍵數據包括總營收、淨利潤，以及它們的同比增長率。並指出主要的收入來源是什麼。
    
3. **資產負債表分析**：公司的總資產、總負債情況如何？現金儲備是否充足？有沒有值得警惕的債務風險？
    
4. **現金流量表分析**：經營活動現金流是正是負？這個數字是增加還是減少了？這說明了什麼？
    
5. **最大的亮點**：從財報中找出 1–2 個最亮眼的積極信號。
    
6. **潛在的風險**：從財報中找出 1–2 個最需要警惕的風險點。    

請以**清晰的、要點式的格式**呈現給我。

prompt
請根據以上內容，規劃一頁互動式圖表網頁呈現重點
